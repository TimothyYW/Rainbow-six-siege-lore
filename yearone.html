  1. Season 1 Operation Black Ice

    As the first season of rainbow bringing two new operators from JTF2 Canda and a new maps. Starting with an attacker named Sébastien "Buck" Côté and a defender named Tina Lin "Frost" Tsang, and the new map named "Yacht".

  2. Season 2 Operation Dust Line

    Operation Dust Line was released on May 10, 2016 which bringing two new operators from Navy Seal America and a new map, an attacker name Craig "Blackbeard" Jenson and a defender name Meghan J. "Valkyrie" Castellano and a map called "Border"

  3. Season 3 Operation Skull Rain

    Operation Skull Rain was released on August 2, 2016 which bringing two new operators from BOPE Brazil and a new map, an attacker name Vicente "Capitão" Souza and a defender name Taina "Caveira" Pereira and a map called "Favela".

  4. Season 4 Operation Red Crow

    Operation Red Crow was released on November 17, 2016 which bring two new operators from SAT Japan, new map, and a map reworke an attacker name Yumiko Imagawa (Japanese: 今川 由美子[1], Imagawa Yumiko), codenamed Hibana and a defender name Masaru Enatsu (Japanese: 江夏 優[1], Enatsu Masaru), codenamed Echo, a map called "Skyscrapper" and a map rework "Barlett University".
